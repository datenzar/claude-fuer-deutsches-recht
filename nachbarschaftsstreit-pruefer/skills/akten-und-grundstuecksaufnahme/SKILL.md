# Akten- und Grundstücksaufnahme

## Zweck

Nachbarrecht steht und fällt mit der Grundstücksrealität. Dieser Skill baut die Tatsachenbasis, bevor Ansprüche behauptet werden.

## Sammelliste

| Unterlage | Warum wichtig? |
|---|---|
| Grundbuchauszug | Eigentum, Dienstbarkeiten, Wegerechte, Grunddienstbarkeit |
| Liegenschaftskarte | Flurstück, Grenze, Lage von Gebäuden |
| Vermessungsunterlagen | Grenzpunkte, Abmarkung, Grenzverwirrung |
| Baulastenverzeichnis | öffentlich-rechtliche Duldungen/Zufahrten |
| Bauakte/Baugenehmigung | genehmigter Baukörper, Abstandsflächen, Auflagen |
| Bebauungsplan/Satzungen | Baurecht, Einfriedung, Baumschutz, Vorgarten |
| Fotos/Videos | Zustand, Zeit, Blickrichtung, Messbezug |
| Korrespondenz | Fristsetzung, Widerspruch, Kenntnis, Eskalation |
| Rechnungen/Kostenvoranschläge | Schaden, Beseitigungskosten, Sicherung |

## Arbeitsprodukt

Erzeuge:

1. **Grundstücksblatt** je Grundstück.
2. **Grenz- und Baukörpermatrix**.
3. **Chronologie** mit Datum, Quelle, Bedeutung.
4. **Lückenliste**.
5. **Beweiskarte**: Foto-ID, Ort, Richtung, Aussage.

## Besonderheiten

- Liegenschaftskarten sind wichtig, ersetzen aber nicht immer eine gerichtsfeste Grenzfeststellung.
- Bei Überbau und Grenzverwirrung früh Vermessung erwägen.
- Bei Wegerechten Grundbuch, Baulasten und tatsächliche Nutzung getrennt prüfen.
