# Quellenanker US Bankruptcy Code

Diese Datei ist ein Arbeitsanker für Livechecks. Für belastbare Ausgaben werden Code-Fassung, Forms, Fees, Local Rules und Docket immer aktuell geprüft.

## Offizielle Quellen

- GovInfo Title 11: https://www.govinfo.gov/app/collection/uscode/2024/title11
- GovInfo Title 11 PDF: https://www.govinfo.gov/content/pkg/USCODE-2024-title11/pdf/USCODE-2024-title11.pdf
- U.S. Code preliminary Title 11: https://uscode.house.gov/view.xhtml?edition=prelim&path=%2Fprelim%40title11
- U.S. Courts Bankruptcy Basics: https://www.uscourts.gov/court-programs/bankruptcy/bankruptcy-basics
- U.S. Courts Bankruptcy Forms: https://www.uscourts.gov/forms-rules/forms/bankruptcy-forms
- U.S. Trustee Program overview: https://www.justice.gov/ust/bankruptcy-fact-sheets/overview-bankruptcy-chapters

## Arbeitsregel

Keine US-Rechtsprechung aus Modellwissen zitieren. Bei jeder konkreten Frist, Fee, Form, Local Rule, Debt Limit, Court Order oder Docket-Frage live prüfen und US-Counsel einbinden.
