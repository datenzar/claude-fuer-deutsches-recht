# Produktbeschreibung VestaCool FR-900

Der VestaCool FR-900 ist ein vernetzter Kühlschrank mit getrennten Temperaturzonen, Eiswürfelmodul, Feuchtesensorik und App-Steuerung. Ohne App ist eine manuelle Grundeinstellung möglich. Nach Händlerangabe funktionieren aber Energieoptimierung, Fehlerspeicher, Wartungsdiagnose, Eiswürfelmodul-Kalibrierung und Warnmeldungen nur über das digitale System.

Technische Bausteine:

| Baustein | Bedeutung |
| --- | --- |
| Firmware Core 5.x | Steuerung Kompressor, Lüfter, Sensorlogik |
| VestaCool App | Temperaturprofile, Update, Warnmeldungen |
| Cloud-Dienst | Diagnose-Token, Ersatzteilfreischaltung, Wartungshistorie |
| NFC-Ersatzteilchip | Pairing von Kompressor und Eiswürfelmodul |
| ServicePort | nur mit Händler-/Herstellerzertifikat vollständig nutzbar |

In der Broschüre findet sich der Satz: "Repair-ready architecture: ServicePort und modulare Bauteile sorgen für schnelle Hilfe." Ein kleiner Sternchenhinweis verweist auf "autorisiertes Servicenetz". Die Mandantin erinnert sich an keine gesonderte Information, dass unabhängige Werkstätten technisch ausgeschlossen sein könnten.
