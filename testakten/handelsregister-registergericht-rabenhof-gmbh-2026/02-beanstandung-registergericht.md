# Schreiben Registergericht

Amtsgericht Charlottenburg - Registergericht
HRB 246810 B

In der Registersache Rabenhof Sensorik GmbH wird um Stellungnahme binnen zwei Wochen gebeten.

1. Die Anmeldung der Kapitalerhöhung verweist auf eine Gesellschafterliste, die prozentuale Beteiligungen nur auf eine Nachkommastelle ausweist.
2. Die Versicherung der neu bestellten Geschäftsführerin enthält keinen ausdrücklichen Bezug auf die Bestellung vom 19.05.2026.
3. Die zur Borchert Venture KG vorgelegten Vertretungsnachweise sind unvollständig.
4. Die Firma enthält weiterhin den Zusatz „Sensorik“, obwohl der Unternehmensgegenstand auch Finanzierungsvermittlung nennt. Es wird um Erläuterung gebeten, ob erlaubnispflichtige Tätigkeiten ausgeübt werden.

Rechtspflegerin R. Kaltenborn
