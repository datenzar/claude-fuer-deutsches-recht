# selbstvertreter-sozialgericht

Ein Plugin fuer Buergerinnen und Buerger, die sich beim Sozialgericht selbst vertreten — ohne Anwalt.

## Fuer wen ist das?

Sie sind versichert oder beziehen Sozialleistungen. Sie haben einen Bescheid bekommen und sind nicht einverstanden. Sie haben keinen Anwalt oder koennen sich keinen leisten. Dieses Plugin hilft Ihnen Schritt fuer Schritt.

## Was leistet das Plugin?

- Sie lernen, wie das Widerspruchsverfahren laeuft.
- Sie lernen, wie eine Klage beim Sozialgericht funktioniert.
- Sie lernen, wann ein Eilantrag noetig ist.
- Sie verstehen die wichtigsten Bereiche: Pflegegrad, Krankenkasse, Buergergeld, Erwerbsminderungs-Rente, Grad der Behinderung.
- Sie sehen, wann Sie doch besser einen Anwalt nehmen sollten.

## Doppelt einfache Sprache

Jede Skill hat einen extra Block "In einfacher Sprache". Dort stehen ganz kurze Saetze. Pro Satz nur ein Gedanke. So koennen auch Menschen mit wenig Lese-Uebung mitkommen.

## Wichtig vorweg

- Vor dem Sozialgericht und dem Landessozialgericht brauchen Sie keinen Anwalt (§ 73 SGG).
- Das Verfahren ist fuer Versicherte und Leistungsempfaenger kostenfrei (§ 183 SGG).
- Das Gericht ermittelt die Wahrheit selbst (Amtsermittlung § 103 SGG). Das hilft Laien.
- Sie koennen die Klage auf der Geschaeftsstelle des Sozialgerichts diktieren (§ 90 SGG).

## Was kann das Plugin nicht?

- Keine richtige Rechtsberatung. Bei kniffligen Faellen brauchen Sie eine Anwaeltin oder einen Anwalt.
- Keine Garantie, dass Sie gewinnen.
- Keine Pruefung Ihres Bescheids im Einzelnen.

Bei Unsicherheit: Wenden Sie sich an eine Beratungsstelle (Sozialverband VdK, SoVD, Diakonie, Caritas) oder beantragen Sie Beratungshilfe.

## Aufbau einer Skill

1. Worum geht es?
2. In einfacher Sprache (3 bis 5 kurze Saetze)
3. Wann brauchen Sie diese Skill?
4. Fachbegriffe
5. Rechtsgrundlagen
6. Schritt-fuer-Schritt-Anleitung
7. Worauf Sie besonders achten muessen
8. Typische Fehler
9. Querverweise
10. Quellen und Aktualitaet

## Stand

Mai 2026. Hinweise zu Reformen sind eingebaut. Bei Unsicherheit pruefen Sie unter www.sozialgerichtsbarkeit.de oder im aktuellen Gesetz.
